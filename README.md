# GRC-Automation

## Overview 

This project is aimed to show my skills in automation. This project will have me redesign a common control to be automated and continuous monitoring. 

## The Scenario

A financial company is using SaaS infrastructure (AWS). The Control I will be looking at is "Privileged Access Review" and this references SOC 2: CC6.1, CC6.2, CC6.3 and PCI DSS: Requirement 7.2, 8.1.4. Currently the manual process is not working and not meeting correct industry standards. While it is smart to review this compliance quarterly or annually, automation allows for this gap to be closes.
